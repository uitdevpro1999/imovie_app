# Routing, DI, and Codegen Skill

## Purpose

Handle generated or registered Flutter infrastructure safely.

## Procedure

1. Detect routing strategy and registration location.
2. Detect dependency injection strategy and registration pattern.
3. Detect generated files and the commands that create them.
4. Change source annotations/configuration only; do not hand-edit generated outputs.
5. Run or document the required generation command.

## Checklist

- Route registration
- Route arguments and result type
- Guard/middleware impact
- DI lifetime
- Constructor injection
- Generated file policy
- Build command and failure handling
