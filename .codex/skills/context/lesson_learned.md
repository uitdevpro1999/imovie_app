# Lesson Learned Skill

## Purpose

Capture reusable learnings so future agents avoid repeated mistakes.

## When To Write

- User correction reveals a missed rule.
- A defect was introduced by the agent.
- A project invariant was discovered.
- A repeated workflow gap appears.
