# Token Efficiency Skill

## Purpose

Use context deliberately so agents can work deeply without loading unnecessary files.

## Rules

- Read summaries, briefs, and manifests before raw source.
- Load only files relevant to the current question.
- Prefer search results and targeted snippets over whole-file dumps.
- Summarize large docs into reusable context files.
- Stop reading when the remaining ambiguity does not affect the next decision.

## Output Discipline

- Keep handoffs compact.
- Link to source paths instead of pasting long content.
- Separate must-know facts from optional detail.
