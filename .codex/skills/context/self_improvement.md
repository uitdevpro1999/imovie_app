# Self Improvement Skill

## Purpose

Run a short improvement loop after important prompts and all agent-caused defects.

## Classification

- No-op: no reusable improvement.
- Backlog: improvement exists but should not block current task.
- Lesson: add a prevention rule or project invariant.
- Patch: update an agent, skill, command, or workflow now.

## Rules

- Agent-caused defect cannot be no-op.
- Prefer small targeted patches.
- Keep global kit improvements generic.
- Log every patch.

## Output

- Classification
- Evidence
- Action taken
- Files changed
- Follow-up if any
